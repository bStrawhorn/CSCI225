Does it start and stall?

Is there spark to the plugs?

Is there fuel to the filter?

Is there spark from the coil?

Is it a mechanical distributor?

Is there 12V+ at the coil primary?

Check condenser, points or magnetic pick-up, rotor, or cap damage.

For electronic distribution, see model manual for diagnostic checks.

Test coil for internal short. check secondary output wire resistance.

Ignition system wiring, voltage regulator

Is it fuel injected?

Vapor lock, fuel pump, blockage.

Single-point, check throttle body. Electronic multi-point, seperate diagnostic.

Try starter spray in carb, throttle open.

Check OBD, blink code.

Ignition timing, fuel problem, cranking too slow- battery, starter.

Does it stall on key release to run?

Read OBD or OBDII or check for blink code access.

Ignition "run" circuit or column key switch failure. Ring out with meter.

Does it stall in rain?

Check for cracked coil, distributor. Check visable electrical arcing running in dark.

Does it stall when warm?

Adjust idle, blow out fuel filter, check furl pump output. Check for vacuum leak or sensor failure.

On cold stalling, check for stuck choke, EGR. Check for vacuum leak.
